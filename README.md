# Mako2Jinja

Converts a given mako template into a jinja template, badly.


	Usage:
	    mako2jinja.py FILE
	    mako2jinja.py (-h | --help)
	    mako2jinja.py --version

	Arguments:
	    FILE                      Input file.

	Options:
	    -h --help                 Show this screen.
	    --version                 Show version.
